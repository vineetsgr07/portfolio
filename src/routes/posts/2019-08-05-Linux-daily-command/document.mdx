#### 1. Find a file 
```jsx

  a) find (the program)
  b) /(the path)
  c) and -name abc.dmg
  find / -name abc.dmg
```

#### 2. To prevent the shell from interpreting the dollar sign, backslash it
```jsx
 
  echo \$remote_add
  or use single quotes
  echo '$remote_addr'
```

#### 3. Execute a .sh file
```jsx

  chmod u+x myfile.sh
  cp myfile.sh /usr/local/bin
  edit ~/.bash_profile and add the following line:
  alias myfile=./myfile.sh
  execute the following command-line:
  source ~/.bash_profile
  then you will be able to run your file as a program
  $ myfile
```

```jsx

  lsof -i :8888

```
